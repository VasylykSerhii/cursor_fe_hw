• git config --global user.name blabla - настройка гита а саме Имя
  • git config --global user.email blabal@bla - настройка гита а саме Email
  • git init - створює пустий репозиторій
  • git status - провірка в якому статусі наші файли
  • git add - додавання нових файлів та змін у вже існуючих файлах в репозиторій
  • git commit - записує файли та зміни в репозиторій
  • git log - історія комітів
  • git checkout - перехід на іншу вітку
  • git branch - перераховую всі створені вітки а також створення нової вітки
  • git push - відправка змін на віддалений репозиторій
  • git pull - получити зміни з віддаленого репозиторія і обєднати із своїми
  • git merge - об'єднання віток
  • git clone - клонування віддаленого репозиторія собі локально

.gitignore - можна описати які файли будуть ігноруватися і не будуть відправлятися на віддалений репозиторій

  • git revert - створення нового коміта який відміняє зміни попререднього коміта
  • git reset - відміняє зміни які ще не відправлені на віддалений репозиторій
  • git rebase - об'єднання віток, але він не зберігає історію комітів як це робить merge
  • git cherry-pick - копіювання коміта із одної вітки в іншу
  • git stash - скриває всі файли які не закомічені і позволяє повернутися до них пізніше
